# Ezh (Ʒ)

Dynamic DRAM optimizations for stencil workloads. A project for [ECE511](https://courses.grainger.illinois.edu/ECE511/sp2026/) at UIUC, spring 2026. 

Named for the letter used to abbreviate the unit [dram](https://en.wikipedia.org/wiki/Dram_(unit)).
